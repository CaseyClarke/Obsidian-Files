# Question 1
the domain traces out the first quarter of the unit circle giving us nice bounds
as $x$ goes from $0 \to 1$ $y$ traces out $y = \sqrt{1-x^2}$ making this $y$ supported thus
$$\iint\limits_D\frac{4y^3}{(x+1)^2}dA =\int_0^1\int_{0}^{\sqrt{1-x^2}}\frac{4y^3}{(x+1)^2}dydx$$
$$\int_{0}^{\sqrt{1-x^2}}\frac{4y^3}{(x+1)^2}dy = \left. \frac{y^4}{(x+1)^2} \right|^{\sqrt{1-x^2}}_0 = \frac{(1-x^2)^2}{(x+1)^2}$$
$$\int_0^1\frac{(1-x^2)^2}{(x+1)^2}dx = \int_0^1\frac{(1-x)^2(1+x)^2}{(x+1)^2}dx = \int_0^1(1-x)^2dx =\left. \frac{-(1-x)^3}{3}\right|^1_0 = \frac{1}{3}$$
$$\iint\limits_D\frac{4y^3}{(x+1)^2}dA = \frac13$$
{{ENDQUESTION}}
# Question 2
line intersects at
$-x = 2x \implies (0,0)$
$2x = 5 \implies (2.5, 5)$
$-x = 5 \implies (-5,5)$
convert from $y$ to $x$ to avoid computing the inner integral in $y$
$x = \frac y2$
$x = -y$
$$\iint\limits_T\cos(y^2)dy = \int_{0}^5\int_{-y}^{\frac y2}\cos(y^2)dxdy$$
$$\int_{0}^5\left. x\cos(y^2)\right|^{\frac y2}_{-y} dy = \int_{0}^5 (\frac y2)\cos(y^2) + y\cos(y^2) dy = \frac32\int_{0}^5 y\cos(y^2)dy$$
$$\frac32\int_{0}^5 y\cos(y^2)dy = \left. 3\sin(y^2)\right|^5_0 = 3\sin(25)$$
$$\iint\limits_T\cos(y^2)dy = 3\sin(25)$$
{{ENDQUESTION}}

$$\left. \frac{y^3}{3} - \frac{2y^5}{5} + \frac{y^7}{7}\right|^{1}_{-1} = \frac13 - \frac25 + \frac17 - (-\frac13 + \frac25 -\frac17) = \frac{16}{105}$$
$$\iint\limits_D4x^3y^2dA = \frac{16}{105}$$
