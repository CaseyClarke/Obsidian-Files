# Question 1

(a)

$-1 \leq cos(x) \leq  1\quad x\in \mathbb{R}$
take $x = n^2 + 1$
$e^{-1} \leq  e^{cos(n^2 + 1)} \leq  e^1$
$\frac{e^{-1}}{\sqrt{n}} \leq  \frac{e^{cos(n^2 + 1)}}{\sqrt{n}} \leq  \frac{e^1}{\sqrt{n}}$
$\lim_{n \to \infty}\frac{e^{-1}}{\sqrt{n}} = 0$
$\lim_{n \to \infty}\frac{e^{1}}{\sqrt{n}} = 0$
by squeeze thm 
$\lim_{n \to \infty}\frac{e^{cos(n^2 + 1)}}{\sqrt{n}} = 0$

(b)

$0 \leq  sin^4(x) \leq  1\quad x\in \mathbb{R}$
take $x = n^3 + 4n + 1$
$0 \leq  sin^4(n^3 + 4n + 1) \leq  1^4$
$0 \leq  \frac{sin^4(n^3 + 4n + 1)}{2^n} \leq  \frac{1}{2^n}$
$\lim_{n \to \infty} \frac{1}{2^n} = 0$
$\lim_{n \to \infty}0 = 0$
by squeeze thm 
$\lim_{n \to \infty}\frac{sin^4(n^3 + 4n + 1)}{2^n} = 0$
