# Question 2

$r'(t) = <e^t, \sqrt{2},-e^{-t}>$
$L = \int_0^1||r'(t)||dt$
$L = \int_0^1\sqrt{(e^t + e^{-t})^2}dt$
$L = \int_0^1e^t + e^{-t}dt$
$L = e^t - e^{-t}|^1_0$
$L = (e^1 - e^{-1}) - (e^0 - e^{-0})$
$L = e - \frac{1}{e}$
