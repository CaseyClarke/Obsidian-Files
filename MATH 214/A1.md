# Question 1

(a)
$a_n = 3^{\frac{-2}{\sqrt{n}}}$
model with $f(x) = 3^{\frac{-2}{\sqrt{x}}}$
$\lim_{x \to \infty}3^{\frac{-2}{\sqrt{x}}} = 3^{\lim_{x \to \infty}\frac{-2}{\sqrt{x}}}$
$\lim_{x \to \infty}\frac{-2}{\sqrt{x}} = 0$
$3^x$ is continuous and the inner limit exists so 
$3^0 = 1$
$\lim_{n \to \infty}a_n = 1$

(b)
$b_n = \frac{7n^4 +2n + 1}{3 - 5n^2 + 2n^4} =  \frac{7 + \frac{2}{n^3} + \frac1{n^4}}{2-\frac5{n^2} +\frac3{n^4}}$
model with $f(x) =  \frac{7 + \frac{2}{x^3} + \frac1{x^4}}{2-\frac5{x^2} +\frac3{x^4}}$
the $x^n$ terms dominate the integer numerators and vanish
$\lim_{x \to \infty}f(x) = \frac{7 + 0 + 0}{2 + 0 + 0} = \frac72$
$\lim_{n \to \infty}b_n =\frac72$

(c)
$c_n  = \frac{n^2+3n}{\sqrt{5+n^3}} = \frac{n^2+3n}{\sqrt{n^3(1+ \frac{5}{n^3})}} = \frac{n+3}{\sqrt{n}\sqrt{1+ \frac{5}{n^3}}}$
model with $f(x) =\frac{x+3}{\sqrt{x}\sqrt{1+ \frac{5}{x^3}}}$
$\lim_{x \to \infty}\sqrt{1 + \frac{5}{x^3}} = 1$
$\frac{x+3}{\sqrt{x}} = \sqrt{n} + \frac{3}{\sqrt{n}}$
$\lim_{x \to \infty}\sqrt{n} =\infty$
$\lim_{x \to \infty}\frac{3}{\sqrt{n}}= 0$
$\frac{x+3}{\sqrt{x}} \to \infty + 0 = \infty$
$\lim_{x \to \infty}f(x) = \frac{\infty}{1} = \infty$
$\lim_{n \to \infty}c_n = \infty$
DNE ($\infty$)

(d)
$d_n = \sqrt{3n^4 + 5n} - n^2 = \sqrt{n^4(3 + \frac{5n}{n^4})} - n^2 = n^2\sqrt{(3 + \frac{5}{n^3})} - n^2$
model with $f(x) =  x^2\sqrt{(3 + \frac{5}{x^3})} - x^2$
$\lim_{x \to \infty}3 + \frac{5}{x^3} = 3$
$\lim_{x \to \infty}x^2\sqrt3 - x^2$
$\lim_{x \to \infty}x^2(\sqrt3 - 1) = \infty$
$\lim_{n \to \infty}d_n = \infty$
DNE ($\infty$)

(e)
$e_n = (-1)^nln(n)$
this obviously diverges so try to prove it
prove that $\exists b_0,b_1$ S.T. There are infinite elements in the series that satisfy each of the following equations
$e_n > b_0$ 
$e_n < b_1$
try $b_0 = 1$ and $b_1 = -1$
the even terms in $e_n$ are in the form
$n = 2k$
$(-1)^{2k}ln(2k) = ln(2k)$
$ln(2k) > b_0 = 1$
$2k > e$
$k > e/2$
there are infinitely many $k$ that satisfy this inequality $\implies$ there are infinitely many $n = 2k$ that satisfy this inequality 
the odd terms in $e_n$ are in the form
$n = 2k+1$
$(-1)^{2k+1}ln(2k+1) = -ln(2k+1)$
$-ln(2k+1) < b_1 = -1$
$2k+1 >  e$
$k > (e-1)/2$
there are infinitely many $k$ that satisfy this inequality $\implies$ there are infinitely many $n = 2k+1$ that satisfy this inequality 
we've now found infinite elements of $e_n$ bigger than $b_0$ and smaller than $b_1$ $\implies$ $e_n$ is not bounded $\implies$ $e_n$ diverges 
basically even subsequence goes to $\infty$ and odd subsequence goes to $-\infty$
DNE (Oscillation between $-\infty$ and $+\infty$)

# Question 3

(a)
$a_n = \frac{2n}{11}\sin(\frac{11}{2n}) = \frac{\sin(\frac{11}{2n})}{\frac{11}{2n}}$
$\lim_{n \to \infty}\frac{\sin(\frac{11}{2n})}{\frac{11}{2n}}$
Let $x = \frac{11}{2n}$
as $n \to \infty$ $x \to 0$
so the limit becomes
$\lim_{x \to 0}\frac{\sin(x)}{x}$
this is a famous limit that equals $1$
$\lim_{n \to \infty}a_n = 1$

(b)
$b_n = n^2\tan(\frac{n-1}{5n^3+2n})$
$b_n = n^2(\frac{n-1}{5n^3+2n})\frac{\tan(\frac{n-1}{5n^3+2n})}{\frac{n-1}{5n^3+2n}}$
Let $x = \frac{n-1}{5n^3+2n}$
$\lim_{n \to \infty}\frac{n-1}{5n^3+2n} = \lim_{n \to \infty}\frac{n(1-\frac{1}{n})}{n(5n^2+2)} = \lim_{n \to \infty}\frac{1-\frac{1}{n}}{5n^2+2}$
$-\frac1n \to 0$ as $n \to \infty$
$\lim_{n \to \infty}\frac{1}{5n^2+ 2} = 0$
so as $n \to \infty$ $x \to 0$ 
$b_n = n^2x\frac{\tan(x)}{x}$
$\lim_{x \to 0}\frac{\tan(x)}{x}$
the above is a famous limit that goes to $1$ so substituting that and back substituting $x$
$\lim_{n \to \infty}b_n = \lim_{n \to \infty}n^2(\frac{n-1}{5n^3+2n})(1)$
$\lim_{n \to \infty}\frac{n^2(n-1)}{n^2(5n+\frac{2n}{n^2})}$
$\lim_{n \to \infty}\frac{n-1}{5n+\frac{2}{n}}$
$\lim_{n \to \infty}\frac{1-1/n}{5+\frac{2}{n^2}}$
the $1/n$ and $\frac2{n^2}$ terms vanish as $n \to \infty$
$\lim_{n \to \infty}\frac15$
$\lim_{n \to \infty}b_n = \frac15$

(c)
$c_n = (1+\frac{5}{n})^{4n} = ((1+\frac{5}{n})^n)^4$
$(1+\frac{c}{n})^n$ is a famous limit that is defined as 
$\lim_{n \to \infty}(1+\frac{c}{n})^n = e^{c}$
so $c_n$ becomes
$\lim_{n \to \infty}c_n = (e^5)^4 = e^{20}$
